# do-test

Experimenting with Cloudflare Durable Objects. Frontend chat shell first; DO wiring later.

## Stack

- React 19
- Vite
- TypeScript 7
- oxlint
- Bun

## Scripts

```bash
bun install
bun dev      # local dev server
bun run build
bun run lint
```

## Layout

Minimal chat UI only — room list, message pane, composer. Local state, no backend.
